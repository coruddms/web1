HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

HTML is a command that defines the font size, font color, font shape, graphics, and document movement (hyperlink) of a document and is used to create a homepage. The commands used in HTML are called tags and are represented by using angle brackets <>".



Generally, tags are made up of two pairs representing the beginning and the end, but they have the effect of starting tags such as "img" and "br".
